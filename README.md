#### ▶  Packages to install

```bash
pip install flask
```



#### ▶ Run commands

```bash
export FLASK_APP=main
```
```bash
flask run
```

#### ▶  Environment

- OS : Windows
- Language : Python 3.9.0
- Libraries :
  - Flask
  - Pillow
  - ImageHash
